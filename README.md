# Point Calculator 4

Une application Meteor 3.4 moderne pour calculer vos points nutritionnels.

## Caractéristiques
- Entièrement migré vers Meteor 3.x (dernière version)
- Utilise le système de modules ES6 (Node 22+)
- Interface moderne avec Google Fonts (Outfit)
- Pas de base de données requise, calculs instantanés côté client

## Installation
Assurez-vous d'avoir Meteor installé, puis :
```bash
meteor npm install
meteor
```
